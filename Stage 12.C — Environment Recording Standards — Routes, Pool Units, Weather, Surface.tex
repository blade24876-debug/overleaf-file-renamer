\subsection*{12.C.1 Environment Unit Concept — Route, Pool, Machine IDs}
\phantomsection
\addcontentsline{toc}{subsection}{12.C.1 Environment Unit Concept — Route, Pool, Machine IDs}

\begin{itemize}
\item Environment Unit IDs are crosswalk keys used to preserve comparability and admissibility for tests.
\item Environment Unit IDs \textbf{MUST} be recorded on Test Event Logs for any test where environment affects validity.
\item A change of environment \textbf{MUST} NOT be treated as equivalent:
\begin{itemize}
\item any substitution requires a new Environment Unit \textbf{ID}
\item baseline comparability must restart under the new \textbf{ID}
\end{itemize}
\item If an admissible test is executed without the correct Environment Unit \textbf{ID}, the test is inadmissible for gate credit and \textbf{MUST} be treated as \textbf{INVALID}.
\end{itemize}

\subsection*{12.C.5 Environment Validity Rules}
\phantomsection
\addcontentsline{toc}{subsection}{12.C.5 Environment Validity Rules}

\begin{itemize}
\item Invalid environment conditions for gate-admissible tests include:
\begin{itemize}
\item wrong or missing Environment Unit \textbf{ID} (\textbf{ROUTE}-##, \textbf{POOL}-##, \textbf{MACH}-## as applicable)
\item pool Lane Length Verified N for any swim test used for gate credit
\item missing machine required settings fields (incline/grade percent, speed + units, resistance level/scale or N/A)
\item unsafe conditions that compromise safety or validity (ice glaze, lightning risk, whiteout/near-zero visibility, official air quality warning, extreme heat posture)
\end{itemize}
\item Mandatory relabeling and baseline restart:
\begin{itemize}
\item any substitution of route, pool, or machine requires a new Environment Unit \textbf{ID}
\item material changes require a new \textbf{ID} and baseline restart, including:
\begin{itemize}
\item construction detours or route closures
\item route surface class change (e.g., asphalt to gravel) when it alters comparability
\item pool lane length change or pool signage inconsistency
\item machine replacement or machine relocation when identity/behavior changes
\end{itemize}
\end{itemize}
\item Weather and hazard posture:
\begin{itemize}
\item unsafe conditions require reslot or substitution per Stage 3 adjustment doctrine
\item do not compare performance across materially different environments without separate IDs and conditions notes
\end{itemize}
\item Treadmill dual-\textbf{ID} rule (binding):
\begin{itemize}
\item treadmill tests \textbf{MUST} log both:
\begin{itemize}
\item \textbf{MACH}-## treadmill unit
\item \textbf{ROUTE}-## treadmill profile descriptor
\end{itemize}
\item missing either \textbf{ID} is an admissibility failure for gate use
\end{itemize}
\item Allowed substitutions rule (binding):
\begin{itemize}
\item no equivalent route substitutions under the same \textbf{ROUTE} \textbf{ID}
\item any substitution requires new \textbf{ID} and new baseline; do not “carry over” comparability claims across different IDs
\end{itemize}
\end{itemize}
